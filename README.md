# Truffictor

Project that help truck driver in avoiding Truck accident. Build using OpenCV, mediapipe, and ESP32 DevKit.

## Installation

1. Clone the Repository
```bash
git clone https://github.com/thheor/truffictor.git
cd truffictor
```

2. Set Up Environment (Optional)
```bash
python3 -m venv venv
source venv/bin/activate # On Linux or MacOs
venv\Scripts\activate # On Windows
```

3. Install Dependencies
```bash
pip install -r requirements.txt
```

*Note: If you are on Linux, you need to install **python3-tk** via your package manager*

## Usage 
Run this in terminal
```bash
python3 app.py
```
